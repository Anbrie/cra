Usage

Don't

Colours should never be used exclusively to convey meaning.

Do

Match colours with textual cues and/or invisible text for screen readers.

Do

Particularly warning and success colours (reds and greens) should always have visible text distinctions, as these appear the same to those with colour blindness.

Do

Colour contrast for text must be at least 7:1 for normal text or 4.5:1 for large text (AAA).

Do

Use a colour contrast checker, such as to test any non-standard colours or colour combinations. Ensure that colours used are WCAG Level AAA.
