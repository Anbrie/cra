Getting a tax return

There are several ways you can get the Income Tax Package for 2020, or the Income Tax and Benefit Package for Non-Residents and Deemed Residents of Canada for 2020. 

Topics

  • Alternate formats for persons with disabilities
  • Which income tax package should you use?
  • Exceptions
  • Prior-year tax return packages
  • Forms and publications for non-residents and individuals who resided in Canada for a portion of the year

Online

You can get the Order alternate formats for persons with disabilities.

You can also use the CRA's online order form to have a printed copy of the Income Tax Package for 2020, and any other forms and publications, mailed to you.

By mail

If you filed your 2019 tax return on paper, you should receive a Income Tax Package for 2020 in the mail.

If you want to file on paper and didn’t receive an Income Tax Package, you can order a copy from the CRA.
