Equal height (Pure CSS)

Equal heights using no script, will render automatically upon page load. Also removes the equal heights for targeted items when panels are no longer in-line (100% width, stacked). Also allows stacked items to become part of the equal height (not available with current design), as long as they are the tallest in the row.

Example one, simple height on panel

CSS attributes are applied to .row (.wb-eqht-css), targeted column grid (.wb-eqht-grd) and .panel (.hght-inhrt-css) divs.

Notes:

  • Using flexbox, items do no wrap by default required flex-wrap:wrap style, but will wrap if no override for mobile, so items need .col-xs-* in order to set the width in mobile (e.g. .col-xs-12 for 100% width)
  • Not optimized in IE11 for mobile settings
    • Need to determine if to leave as is (since there is no IE11 mobile browser), or
    • remove styling when columns reach col-xs-12 using browser targeted CSS
